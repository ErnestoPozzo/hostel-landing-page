This project is a **Next.js landing page** built as part of a portfolio showcasing modern web development practices. It uses **TailwindCSS** for styling and integrates **CI/CD pipelines** for seamless deployment.

⚠️ **Note:** The site is currently under construction and not yet finished. Features and design are actively being developed.

---

## 🚀 Features

Clean and responsive design with TailwindCSS

Optimized Next.js setup for fast performance

ESLint + Prettier for consistent code quality

CI/CD pipeline for automated deployment

Hosted on Vercel for reliable, scalable, and fast delivery

## 🧪 Playground Folder

The /playground directory is a sandbox environment used for quick experiments, scratch code, and testing ideas.

- Ignored by Git: The folder is excluded via .gitignore, so files inside will never be committed or pushed.

- Safe for experiments: You can freely create, modify, and delete files here without affecting the production codebase.

- Consistent tooling: ESLint and Prettier configurations from the root project still apply, ensuring code inside /playground follows the same formatting and linting rules.

## 📦 Installation

```
npm install
npm run dev
```

### 🛠 Development

Run linting:

```
npx eslint .
```

Format code:

```
npx prettier --write .
```
